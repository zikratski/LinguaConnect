Videoroom API

API docs by Redocly

API каталога видеокомнат (1.0.0)

Download OpenAPI specification:Download

API для управления главным экраном платформы видеокомнат для проведения языковых клубов.

Получение списка видеокомнат

query Parameters
cursor
string

Курсор для получения следующего набора видеокомнат

limit
integer [ 1 .. 100 ]
Default: 10

Максимальное количество видеокомнат для возврата

language
string
Enum: "EN" "RU" "ES" "FR" "IT" "PT" "ZH" "JA" "AR" "HI" "KO"

Фильтр по языку видеокомнаты

level
string
Enum: "Beginner" "Intermediate" "Advanced"

Фильтр по уровню языка

min_participants
integer >= 0

Минимальное количество участников видеокомнаты

max_participants
integer >= 0

Максимальное количество участников видеокомнаты

Responses

https://zikratski.github.io/conferences
